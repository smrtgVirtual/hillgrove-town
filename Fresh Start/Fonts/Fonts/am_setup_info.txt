Hello! This package contains traffic signs used in Poland.
Files are described as follows:
files named a1.sco etc. are normal plates
files named a1s.sco etc. are signs joined with pole
slup00 and slup01 are poles which have attachment points.
slup00 AttPntNr are 0 or 1
slup01 AttPntNr are 0, 1 and 2

Beta 0.4.0 changes:
- Added font for signs (tablica.oft)
- Added editable signs ("E and T" category)
- Added "G" category signs

Beta 0.3.0 changes:
- Added "T" category signs, with few exceptions
- Editable "T-1" sign, for testing purposes

Beta 0.2.0 changes:
- Added "D" category signs
- Added [attach] parameters into poles

Beta 0.1.0 changes:
- Added "C" category signs and one from "D" category
- Added *.sco files for 40kmh speed restriction 
- Added better readme and licence.
